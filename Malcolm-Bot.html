HOW MALCOM CAN HELP YOU MASTER NEW SKILLS

Open slack bot. Tell bot which skill you are working on for current session -- HTML, CSS, d3, etc. Specify how many cumulative hours of practice you want to complete over time. Bot tracks activity as you work. Optional pomodoro clock setting. Alerts you every 25 minutes to take a break. Another option: suggests activity to do during break--snack, go for a walk, listen to music, meditate, check messages, etc. Provides summary of coding activity at end of session. Stores all activity in a spreadsheet. Also, uses data to create chart showing amount of time spent on a skill. Sends an alert (congratulatory message???) once time goal been reached.
